## file explorer
```
% -- new file
d -- new dir
R -- rename
D -- delete
- -- up directory
:split . or :Sex -- dir of current file
:vsplit . or :Vex -- dir of current file
:sp. -- working dir
:vs. -- working dir
```

## etc
```
:%!<command> -- pipe buffer to external command and replace with command output
:w !<cmd> -- write buffer to command
```
